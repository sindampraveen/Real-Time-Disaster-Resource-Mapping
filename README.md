# Real-Time Disaster Resource Mapping with SOS Coordination and Volunteer Integration

Overview

A web-based disaster management system that helps coordinate emergency resources, SOS requests, and volunteers in real time.

Features

- SOS Alert Management
- Volunteer Coordination
- Resource Tracking
- Real-Time Location Mapping
- Emergency Response Support

Technologies Used

- Python
- Django
- ReactJS
- SQL
- HTML
- CSS
- JavaScript

Project Type

Team Project developed as part of our college curriculum.

Team Contribution

This project was completed collaboratively by our team, and I actively participated in the project development and implementation.
